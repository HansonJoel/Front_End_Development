Basic function

A JavaScript function is a block of code designed to perform a particular
This provides a way of organising and reuse your JavaScript code as you wouldn't need hundreds of the same statement all over the place doing the same thing. This will make your code unreadable.

What differentiate a function from a normal statement is that, functions by itself will not do anything unless we call the function.
To call a function, we simply need to get out of the function, and type in the function name folowed by parenthesis();.
see example in the script section. saysomething();

Why Functions?

  1. With functions you can reuse code
  2. You can write code that can be used many times.
  3. You can use the same code with different arguments, to produce different results.
