String

The String type represents a sequence of characters forming a piece of text. Common operations on texts include concatenation, extraction of the substring, and checking the length of the string. Strings are extensively used in programming and even more so in web development, as both HTML and a big part of Internet content is text. The most common use of text in web development includes: links and paths to resources; tokens; checking user-filled forms and input; dynamic content generation Strings, like other primitives, are immutable, so when we want to change even one letter in a string, in reality, we create a new string.
