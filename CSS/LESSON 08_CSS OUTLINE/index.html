CSS OUTLINE

An outline is a line drawn outside the element borders to make the element outstanding

CSS has the following outline properties:

  • outline-styles
  • outline-color
  • outline-width
  • outline-offset
  • outline

Outline differs from borders. It is drawn outside the element's border and may overlap other content.
Also, the outline is not part of the elemnet dimension as the element width and height is not affected by the width of the outline
