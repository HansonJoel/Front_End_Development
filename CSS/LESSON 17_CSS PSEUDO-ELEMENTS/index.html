CSS PSEUDO-ELEMENT

CSS pseudo-element allows you to style specific parts of an element without adding extra HTML markup. They are prefixed with two colons (::)
The two kinds of Pseudo element are:
::before and ::after
Pseudo-element can be used to:

  • Style the first letter or line, of an element
  • Insert content before or after an element
  • Style the markers of list items
  • Style the viewbox behind a dialog box
