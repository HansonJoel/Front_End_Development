CSS padding

Section 1: This is a solid border

Padding is used to create space around element content, inside any defined borders
.There are different properties for speecifying the padding for each side of an element. They include

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Note: None of the other CSS border property will have effect unless the border-style property is set
