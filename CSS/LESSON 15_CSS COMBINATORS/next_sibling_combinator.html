Next Sibling Combinator (+)

Next Sibling Combinator also known as Adjacent Sibling Selector, Selects an element that is immediately after another specific element (siblings with the same parent).
Syntax: selector1 + selector2
h2 + p: This selects the first <p> element that comes immediately after any <h2> element.
