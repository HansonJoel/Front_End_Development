CSS Pseudo-Classes

Pseudo-classes are keywords added to selectors that specify a special state of the selected element(s). They allow you to style elements dynamically based on user interaction, element position, or other factors.
For Example, it can be used to:

  • Style an element when a user moves the mouse over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus
  • Style valid/invalid/required/optional form elements
