CSS DISPLAY

The CSS display property is one of the most fundamental and powerful tools for controlling the layout of elements on a webpage. It determines how an element is rendered in the webpage and how it interacts with other elements.b
The most common display values are:

Value Description
block Takes full width, starts on a new line
inline Takes only necessary width, flows in line with text
inline-block Mix of inline and block (flows in line but respects width/height)
none Element is hidden and removed from layout
flex Enables flexible box layout (modern 1D layout)
grid Enables CSS Grid layout (modern 2D layout)
