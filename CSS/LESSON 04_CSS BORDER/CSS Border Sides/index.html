Border Colors

Section 1: This is a solid border

The Border Colors specifies the kind of border to display. There are difference types of borders styles. They include

  • solid
  • dotted
  • double
  • groove
  • ridge
  • inset
  • outset
  • none
  • hidden
  • dotted

Note: None of the other CSS border property will have effect unless the border-style property is set
