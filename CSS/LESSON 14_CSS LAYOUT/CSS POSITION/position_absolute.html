Position:absolute

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed). See Example1

So, what does “nearest positioned ancestor” mean?

It's the closest parent (or ancestor) that has a position set to relative, absolute, or fixed.

However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling. See Example2

it is similar to float element however, there is a key difference which is absolute element does not respect boundaries whereas float respects boundaries i.e absolute position element are removed from the flow. absolute position element does not respect the flow and can go on top of any element

Setting the top, right, bottom, and left properties are used to position the element.
