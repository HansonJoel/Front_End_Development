BACKGROUND IMAGE

Image Background

The background image property specifies an image to use as the background of an element. By default, the image is repeated so as to cover the entire element


Background Repeat

By default, the backgroung image property repeats an image both horizontal and vertically and this may look strange. You can use the background-repeat property to spaecify how the image is repeated in the background


Background Position

The background position is used to specify the position of the background image


Background size

The background size is used to specify the size of the background image


Background Attachment

The background attachment property specifies whether background image should scroll or be fixed (which will not scroll of page) i
