Section 1: This is a solid border

CSS Margins are used to create space around elements outside of any defined borders.

There are properties for setting the margins for each side of an element. There are:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

The auto value is used to horizontally center the element within its container.
The element will then take up the specified width and the remaining space will be split equally between the left and right margins
